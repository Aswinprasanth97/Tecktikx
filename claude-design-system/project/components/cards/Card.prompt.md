Surface container. Depth comes from surface color, not shadow. Alternate cream `feature` and `dark` cards band-to-band; reserve `coral` for full-bleed callout moments.

```jsx
<Card variant="feature">…feature copy…</Card>
<Card variant="outline">…pricing tier…</Card>
<Card variant="dark">…product mockup…</Card>
<Card variant="coral" pad="xxl">…CTA…</Card>
```

`pad`: none / lg / xl (default) / xxl / band. `radius`: md / lg (default) / xl.
